### 已知限制

1. **Transformers 版本旧**：评测时 `device_map` 不支持 `"cuda:0"`，已修改 `scripts/evaluation/evaluate.py` 为 `"auto"`
2. **磁盘空间紧张**：可用仅 ~2GB，checkpoint 总量需控制
3. **PyTorch < 2.6**：断点续训时 `optimizer.pt` 加载会失败，需用 `--no-random-init` + `--model-id` 方式

## 2. 数据准备

```bash
# 从 HuggingFace 下载 Chronos 官方语料子集（TSMixup : KernelSynth = 9:1）
# 注意：100 万条约需 2~5 GB 磁盘空间
python scripts/data/prepare_chronos_corpus_subset.py \
  --output_dir data/processed \
  --data_level v100 \
  --min_length 576 \
  --max_length 2048 \
  --seed 42 \
  --compression lz4
```

产出文件：
```
data/processed/chronos_tsmixup_900k.arrow      # 90 万条
data/processed/chronos_kernel_synth_100k.arrow # 10 万条
data/processed/chronos_corpus_subset_summary.json
```

## 3. 训练

```bash
# Tiny 主实验（50K steps，~10h V100）
python scripts/training/train.py \
  --config scripts/training/configs/chronos_t5_tiny_v100.yaml

# 或用一键脚本
bash scripts/cloud/tiny_v100_train.sh

# Mini 探针实验（10K steps，~2h）
python scripts/training/train.py \
  --config scripts/training/configs/chronos_t5_mini_t4_10000.yaml
```

训练产出（以 Mini run-10 为例）：
```
outputs/checkpoints/chronos_t5_mini_t4_probe/run-10/

│   ├── config.json
```

监控命令：

## 4. 评测

### 方式 A：完整评测脚本（多数据集 CSV）

```bash
# 评测指定 run 的 checkpoint-final，覆盖 quick-zero-shot + quick-in-domain
python scripts/evaluation/run_checkpoint_evaluation.py \
  --run_dir outputs/checkpoints/chronos_t5_mini_t4_probe/run-10 \
  --model_name chronos_t5_mini_run10 \
  --checkpoints checkpoint-final \
  --suites quick-zero-shot,quick-in-domain \
  --device cuda:0 \
  --torch_dtype float16 \
  --batch_size 8
```

可选评测套件（`--suites`）：
| 套件 | 数据集数 | 序列总数 | 耗时 |
|------|:--:|:--:|:--:|
| quick-zero-shot | 5 | ~28K | ~30min |
| quick-in-domain | 10+ | ~30K | ~45min |
| zero-shot | 27 | ~85K | 2h+ |
| in-domain | 15 | ~86K | 2h+ |

### 方式 B：快速单点评测（仅 MASE/WQL，~30 秒）

```bash
python scripts/evaluation/quick_eval.py \
  --checkpoint outputs/checkpoints/chronos_t5_mini_t4_probe/run-10/checkpoint-final \
  --n_series 200
```

实时输出：
```
========================================
  Series evaluated: 200
  MASE = 1.8821
  WQL  = 0.1702
========================================
```
